
Social media analytic tool (Project) (Python)
• Designed a social media analytics system using object-oriented programming in Python  

• Implemented a core class hierarchy for multiple profile types with operator overloading to merge profiles of the same type  

• Stored profiles in a fully recursive Binary Search Tree indexed by profile ID  

• Tracked metrics such as followers, likes, and group memberships per profile  
• Validated functionality using black-box and glass-box testing with PyTest
